# Have a good day landing

## This is a simple adaptive landing

Demo you can see, if you click [here].

### Technologies that were used in the project:
+ HTML
+ SASS -> CSS
+ JS
+ Bootstrap 4

## How to start project
 - open console 
 - `git clone https://github.com/scandy5/DSGN_landing`
 - `npm i`
 - `gulp`
